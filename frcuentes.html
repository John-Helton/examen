Preguntas frecuentes sobre Bootstrap


¿Qué beneficios me aporta utilizarlo?

por @tutobootstrap

Buenas prácticas: Bootstrap es un ejemplo bastante bueno de utilización de tecnologías como HTML5, LESS, CSS3, Responsive Web Design y jQuery. Como disponemos de su código fuente, resulta un auténtico filón para aprender buenas prácticas en CSS, si no somos ordenados y seguimos buenos patrones, estaremos más predispuestos a lo que se llama Spaguetti Code (donde vas haciendo "corta y pega" de trozos de código generando al final un ficheros con una estructura desordenada y compleja). Resulta muy recomendable estudiar como está organizado el código.

¿Se puede extender Bootstrap?

por @tutobootstrap

Por supuesto. De hecho el éxito de este proyecto no está sólo en lo que ofrece, sino que además se ha convertido en todo un referente a la hora de desarrollar interfaces de usuario. Por ello, la comunidad open source se ha volcado con él poniendo a nuestra disposición Enlaces y recursos que amplian y mejoran las posibilidades, abriendo un abanico más que interesante.

¿Bajo qué licencia se distribuye?

por @tutobootstrap

Bootstrap está distribuido bajo licencia Apache License v2.0. En maestros del web nos explican claramente el tema de las licencias.

Podemos hacer aplicaciones web cumpliendo con la licencia sin ningún problema. De hecho hay una gran comunidad que utiliza bootstrap para sus desarrollos, y mucha gente está ganando dinero utilizando bootstrap como framework base.
